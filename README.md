# Portfolio Website

This portfolio website showcases my web-development and programming projects. It uses JavaScript objects, arrays, JSON, sessionStorage, localStorage, and DOM manipulation to dynamically generate project cards and interactive page features.

## Languages and Technologies

- HTML
- CSS
- JavaScript
- JSON
- DOM Manipulation
- sessionStorage
- localStorage
- Git
- GitHub Pages

## Features

- Dynamically generated project cards
- JavaScript objects stored in an array
- Project data saved with `JSON.stringify()`
- Project data stored in `sessionStorage`
- Stored project data retrieved with `JSON.parse()`
- Dark-mode preference stored with `localStorage`
- Welcome modal
- Timed notification message
- Contact-form confirmation message
- Responsive project-card layout

## Dependencies

This project does not require any external libraries or frameworks. It runs using standard HTML, CSS, and JavaScript in a modern web browser.

An internet connection is needed to:

- View placeholder project images
- Open GitHub repository links
- Access the published GitHub Pages website

## Running the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a modern web browser.

The project can also be viewed through GitHub Pages after it is published.
